Nada é guardado — introduz os dados sempre que abrires a página.
A chave é usada apenas no browser para chamar diretamente a API gratuita da Groq (Llama 3.3 70B), não é enviada a nenhum outro servidor. Obtém uma key grátis em console.groq.com/keys (sem cartão de crédito). Todas as consultas Cypher — do grafo visual e do chat — são só de leitura (MATCH/RETURN); nunca é escrito na base de dados. As tuas conversas com o chatbot ficam guardadas neste browser (localStorage), isoladas por projeto, e voltam a aparecer no menu lateral da próxima vez que abrires esta página — nunca são enviadas para lado nenhum. A Tavily key (opcional, grátis em tavily.com) ativa pesquisa na internet; sem ela, o chatbot responde só com o que sabe e com o grafo. O agente/ferramenta certa (grafo, pesquisa, calculadora, meteorologia ou resposta geral) é escolhido automaticamente pelo router para cada pergunta — não precisas de selecionar nada.
Liga-te ao Neo4j Aura para visualizar o grafo.
Clica num nó ou relação para ver os detalhes. Duplo-clique num nó expande os seus vizinhos diretamente no grafo.
Dica: devolve nó, relação e nó (ex.: RETURN n, r, m) para a melhor visualização · duplo-clique num nó para expandir vizinhos · passa o rato para realçar ligações · clica na legenda para mostrar/ocultar tipos de nó.
